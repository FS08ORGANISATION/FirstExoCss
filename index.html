Au cours de cette formation nous allons avoir 2 matières:

L'intégration

Le HTML:

Le HTML sert à créer le contenu du site: il va contenir uniquement la partie textuelle du site

Le CSS:

Le CSS va nous permettre d'ajouter la couche graphique à notre site: les couleurs, les formes, le fond etc

Le developpement

Le JavaScript:

Le JavaScript est un langage de programmation de script employé pour rendre une page web interactive: le comportement de la page et son apparence pourront varier en fonction de certains paramètres

Le php:

Le php est un langage de programmation libre, utilisé à gérer des pages web dynamiques qui sont donc modifiés en fonction de certaines informations (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc)

MySQL est un système de gestion de base de données, il permet d'enrégistrer des informations remplies par l'utilisateur (exemple: formulaire de création de compte) pour ensuite être réutilisé au moment approprié (exemple: connexion à un site internet avec identifiant et mot de passe)
